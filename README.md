# Ashmitha P R - Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and experience as a software developer and AI enthusiast. The website is built using modern web technologies and features a responsive design, smooth animations, and interactive elements.

## Features

- **Responsive Design**: The website is fully responsive and works seamlessly on all devices, from desktops to mobile phones.
- **Smooth Animations**: Utilizes CSS and JavaScript for smooth scrolling, fade-in effects, and hover animations.
- **Interactive Elements**: Includes interactive buttons, progress bars, and a custom cursor effect.
- **Project Showcase**: Displays my top projects with links to their GitHub repositories.
- **Experience Timeline**: A visually appealing timeline to showcase my professional experience.
- **Contact Form**: A functional contact form to get in touch with me.

## Technologies Used

- **HTML5**: For structuring the content of the website.
- **CSS3**: For styling and animations, including custom gradients and transitions.
- **JavaScript**: For interactive elements, smooth scrolling, and form validation.
- **Font Awesome**: For icons used throughout the website.
- **Google Fonts**: For typography, using fonts like Poppins, Montserrat, and Playfair Display.

## Project Structure

- **`index.html`**: The main HTML file that structures the content of the portfolio.
- **`style.css`**: The CSS file that contains all the styles, animations, and responsive design rules.
- **`script.js`**: The JavaScript file that handles interactive elements, smooth scrolling, and form submission.
